Question:  When should accessor methods be used?

Answer: 
- When you want to provide an abstract interface or hide the internal representation of an object.
- When we need to enforce some rules on the value being set.
- When we need to provide a read-only access to a property or attribute.

Question: If you create a constructor method to build a DVD by title then create a 
constructor method to build a DVD by category. Does JAVA allow you to do this?\

Answer: Yes, Java allows to create multiple constructor methods for a class, as long as 
they have different parameter lists. This is known as constructor overloading.
